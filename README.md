# 💠 KENG CUSTOM | CORE PC STORE

โปรเจคเว็บแอปพลิเคชัน Full-Stack สำหรับจัดสเปกและจำหน่ายอุปกรณ์คอมพิวเตอร์แบบครบวงจร ออกแบบด้วย UI/UX ธีม Cyberpunk ช่วยเหลือเบื้องต้นในการตรวจสอบความเข้ากันได้ของฮาร์ดแวร์


---

## ✨ ฟีเจอร์หลัก (Key Features)

### 🛠️ 1. ระบบจัดสเปกคอมพิวเตอร์อัจฉริยะ (Smart PC Builder)
* **Real-time Calculation:** เลือกอุปกรณ์แล้วคำนวณราคารวมให้อัตโนมัติทันที
* **Smart Compatibility Check:** ระบบแจ้งเตือนอัจฉริยะ หากเลือก CPU และ Mainboard ที่ Socket ไม่ตรงกัน (เช่น เอา CPU LGA1700 ไปใส่บอร์ด AM5 ระบบจะบล็อกและแจ้งเตือนทันที)
* **Wattage Estimator (คำนวณการใช้ไฟ):** รวมค่าพลังงาน (TDP) ของอุปกรณ์ที่เลือก และแจ้งเตือนหากเลือก Power Supply (PSU) ที่จ่ายไฟไม่เพียงพอต่อระบบ
* **Auto-Save:** ระบบบันทึกสเปกคอมที่จัดค้างไว้ลง Local Storage อัตโนมัติ ปิดเว็บแล้วเปิดใหม่ข้อมูลก็ยังอยู่ครบ
* **Export & Share Spec:** สามารถกดปุ่ม "Share Spec" เพื่อดาวน์โหลดสเปกที่จัดไว้ ออกมาเป็นไฟล์รูปภาพ (`.png`) ดีไซน์สวยงาม พร้อมแชร์ให้เพื่อนดูได้ทันที

### 💳 2. ระบบสั่งซื้อสินค้า (Secure Checkout)
* **Cart Integration:** ดึงข้อมูลสินค้าที่ผู้ใช้จัดสเปกไว้มาแสดงในหน้าชำระเงินอัตโนมัติ
* **Shipping Form:** ฟอร์มกรอกข้อมูลที่อยู่สำหรับจัดส่งสินค้า
* **Payment Methods:** รองรับการเลือกวิธีชำระเงินหลากหลายรูปแบบ (QR พร้อมเพย์, บัตรเครดิต, เก็บเงินปลายทาง)
* **Order Summary:** สรุปยอดรวมสินค้าและคำนวณค่าจัดส่งแบบ Real-time

### 🛡️ 3. ระบบจัดการหลังบ้าน (Admin Dashboard)
* **Inventory Management:** ระบบแอดมินสำหรับจัดการสต็อกสินค้า เพิ่ม/ลด จำนวนสินค้าในคลังได้ในคลิกเดียว (Stock IN/OUT) แบบไม่ต้องรีเฟรชหน้าเว็บ
* **Product CRUD:** สามารถจัดการฐานข้อมูลสินค้าผ่าน API ที่สร้างขึ้นมาโดยเฉพาะ

### 🌐 4. ระบบการแสดงผลและ UI/UX (User Experience)
* **Multi-language Support:** รองรับการเปลี่ยนภาษา (EN / TH) ได้แบบ Real-time
* **Search & Sorting:** ระบบค้นหาสินค้าจากชื่อ/แบรนด์ และการเรียงลำดับราคาสินค้า (มากไปน้อย / น้อยไปมาก)
* **Cyberpunk UI:** ดีไซน์โหมดมืด (Dark Mode) ตัดกับแสงสีฟ้า Neon Cyan ล้ำสมัย และรองรับการแสดงผลทุกหน้าจอ (Fully Responsive)

## 🛠️ เครื่องมือและเทคโนโลยีที่ใช้ (Tech Stack)

* **Frontend:** Next.js 14+ (App Router), React, Tailwind CSS
* **Backend:** Next.js API Routes (สร้าง RESTful API)
* **Database:** SQLite (สำหรับ Local/Demo) จัดการโครงสร้างผ่าน Prisma ORM
* **Libraries:** * `react-hot-toast` (สำหรับป๊อปอัปแจ้งเตือน)
  * `html-to-image` (สำหรับแปลงโค้ด HTML เป็นรูปภาพตอนแชร์สเปก)
